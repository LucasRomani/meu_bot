Instalar o Python:

Descarregue o Python no site oficial (https://www.python.org/downloads/).

⚠️ MUITO IMPORTANTE: Na primeira tela de instalação, marque a caixa que diz "Add Python to PATH" (ou "Adicionar Python ao PATH"). Se não marcar isto, o computador não saberá como rodar o robô.

Crie e salve o conteudo a seguir em um arquivo .bat.

@echo off
title Instalador de Dependencias - Bot Sischef e QRPedir
color 0A

echo ===================================================
echo     Instalando Bibliotecas do Bot Sischef/QRPedir
echo ===================================================
echo.

echo Verificando se o Python e o PIP estao instalados...
python --version
if %errorlevel% neq 0 (
    echo [ERRO] Python nao encontrado! 
    echo Por favor, instale o Python do site oficial (python.org).
    echo ATENCAO: Durante a instalacao, marque a caixa "Add Python to PATH"!
    echo.
    pause
    exit
)

echo.
echo Atualizando o gerenciador de pacotes (PIP)...
python -m pip install --upgrade pip

echo.
echo Instalando as bibliotecas necessarias...
pip install pandas requests selenium webdriver-manager openpyxl

echo.
echo ===================================================
echo     Todas as instalacoes foram concluidas!
echo     Ja pode executar o seu robo.
echo ===================================================
pause
